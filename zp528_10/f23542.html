アドニスMP-MIX3 （MPMIX3） タンデムアンプ用ミキサー【ゆ】

☆音楽を聴きながら、さらにバイク用ナビやレーダーの音声を聞きたい時のタンデムアンプ用のミキサー
MP-MIX3をタンデムアンプの外部入力端子に接続して、オーディオ機器を接続し音楽を聴くことや、さらに、バイク用ナビやレーダーを接続すれば、それらの音声をミックスして聞くことができます。

MP-MIX3の入出力（3.5mmステレオ）
●入力A: プラグ付コード(コード長[約1.5m] ) オーディオ機器・ナビゲーターおよびレーダー他
●入力B: プラグ付コード(コード長[約1.5m] ) オーディオ機器・ナビゲーターおよびレーダー他
●入力C: プラグ付コード(コード長[約1.5m] ) オーディオ機器・ナビゲーターおよびレーダー他
●出力 : ジャック付コード(コード長[約1.5m] ) タンデムアンプの外部入力端子へ

●形状(突起部含まず):[約30(W)×20(H)×40(D)mm] 本体のみ防水構造
●重量:約150g
※ご注意
・ 本機にオーディオ機器・ナビゲーターおよびレーダーは含まれておりません
・ 本機に接続されたタンデムアンプのスピーカーからはモノラルで聞こえます
