『ガイノイドTalk flower』は、少年のような、憂いを帯びた中性的な声が特徴のキャラクター。

[ガイノイドTalkとは]
株式会社エーアイが開発した、高性能な音声合成「AITalk」を搭載したソフトウェアです。
予め録音された、膨大な音声データベースを利用して、非常に人間的で自然な音声合成を実現します。

『ガイノイドTalk flower』の特徴
・テキストボックスに文字を入力するだけの簡単操作！
・グラフィカルで分かりやすいインターフェースで、イントネーションの細かい調整が可能
・辞書登録機能を使って、よく使う単語を予め登録しておくことが可能
・キャラクターデザインは「△○□×(みわしいば)」が担当

[動作環境] Windows対応
・Windows : Windows 8.1, Windows 10 (32bit/64bit対応、最新のWindows Updateが適用されていること)

・CPU : Intel / AMD Dual Core 以上のプロセッサー (Intel Core i3 以上)
・メモリ : 4GB以上推奨(最低2GB)
・ストレージ : 1.5GB以上の空き容量
・ディスプレイ解像度 : XGA(1024x768) 以上
・その他 : DVDドライブ、DirectX 9.0c 以降に対応したオーディオデバイス

※本製品を利用するためには、インターネットに接続できる環境が必要です
※日本語OS以外のOSは動作保証外です
※制作した音声を音声ファイルとして保存するためには、より多くの空き容量が必要です
※ガイノイドTalkのご利用には、.NET Frameworkのインストールが必要です(ガイノイドTalkインストール時に、自動的にインストールされます)
※Virtual PC等の仮想環境上での動作は保証いたしません
※上記動作環境を満たしている場合でも、全てのコンピュータに於ける動作を保証するものではありません
