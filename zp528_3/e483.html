CORSAIR iCUE COMMANDER CORE XT スマート RGB ライティングとファンスピードコントローラー CL-9011112-WW

CORSAIR iCUE COMMANDER CORE XTは、CORSAIR iCUEソフトウェアを使用して、ケースをスマートケースに変えます。ケースファンは最大6個と複数のRGBコンポーネントのデュアル照明チャンネルを便利に制御できます。
最大6個のPWMファンに電力を供給し、制御し、ファンの速度を正確に調整します。ゼロRPMモードを含む、低負荷でもノイズを抑えます。
箱から出して最大264個のRGB LEDをコントロール ? 最大6個のCORSAIR RGBファン、さらに追加の照明チャネルを接続して、Hydro Xシリーズ照明、RGB LED Lighting PRO拡張キットライトストリップ、RGBファンLEDハブなど他のCORSAIR RGBデバイスを制御します。
COMMANDER CORE XTは、高性能ML PRO RGBや超鮮やかなQL RGBなど、最大6個の個別アドレス可能なCORSAIR RGBファンの任意の組み合わせをインテリジェントに検出し、これまで以上に簡単にセットアップできます。
付属の位置変更可能な温度センサーが2つ付いているので、CORSAIR iCUE ソフトウェアでシステム全体の温度を正確に監視できます。
USB 2.0パススルーポートにより、CORSAIR AXiシリーズPSUやPRO XTクーラーなどの内部USBデバイスを接続できます。
カスタムファンカーブと照明プロファイルを設定し、システム温度に基づいてファンの速度とRGB照明を自動的に調整します。
強力なCORSAIR iCUEソフトウェアは、RGB照明のカスタマイズ、CPUとクーラント温度の監視、ファンの速度調整、ファンのカーブのセットアップのための単一のインターフェースを提供します。
埋め込みマグネットと薄型デザインにより、COMMANDER CORE XTをケース内の平らな表面に簡単に取り付けることができます。

CORSAIR iCUE COMMANDER CORE XT は、最大で 6 個の PWM ファンと複数の RGB コンポーネントをインテリジェント制御しながら、システム温度の正確な監視もでき、すべて一つのデバイスで行えます。
