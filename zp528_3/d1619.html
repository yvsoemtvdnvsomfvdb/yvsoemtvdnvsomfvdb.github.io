主な仕様 【7 IN 1 USB Type C ハブ】本HDMI USB Cドックは、単一のUSB-C接続でイーサネット、USB 3.0、HDMI & DP & VGA、および100WのPD充電機能を追加します。本USB Cドッキングステーションは、USB Type-C又はThunderbolt 3 ポートを搭載したパソコンに対応します。本USB C HDMI DisplayPort VGA変換アダプタは、USB-Cでビデオを表示するためにDisplayPort Alternate Modeに対応し必要があります。br【トリプルディスプレイ対応】USB-Cハブで一台又は複数の外部ディスプレイを接続します。 本type-c ハブは3840x2160@60Hz HDMI/3840x2160@60Hz DisplayPort/1920x1080@60Hz VGAのビデオ解像度を選択でき、HDMI/DP/VGA経由で同時に最大3台の1920x1080@60Hz外部ディスプレイを接続できます。 本TypeC ハブを使用して、何時でもUHDテレビ、モニター、またはプロジェクターを簡単に接続できます。br【100W PD充電対応】USB Type-C ハブはPower Delivery機能に対応、パソコンの使用中に最大100ワットで充電できるだけではなく、USBポートに接続されたデバイスも急速充電でき、一つのUSB-C又はThunderbolt 3 ポートの機能を最大化にして、パソコンの充電中にお気軽に複数のデバイスを接続できます。 このバス駆動型のType C ハブで、USB-Cチャージャーを接続しなくても正常に使用可能です。br【多機能USB C ドッキングステーション】このUSB Type-C ハブを用いると、ワイヤレスマウス又はキーボードとの接続に対応され、ドライバーが不要で1000Mbpsギガビットイーサネットネットワークと接続でき、 Cable Matters EZ-Dock ユーティリティ ソフトウェア (Windows) で MAC アドレス パススルー (MAC クローン) をサポートし、フラッシュドライブへのデータ転送に対応され、ワークスペースの拡張にDisplayPort、HDMI、または VGA ディスプレイと接続できます。br【USB Cアダプターハブ】本 type-c ハブは頑丈なデザインとカスタムケースを備えて、優れた耐久性と信頼性を確保でき、外出先や自宅ですべての機能をお手軽にセットアップできます。 このUSB Type-C ハブを用いると、ンパクトサイズで持ち転びに便利な設計により、出張、旅行など外出中に大活躍です。
