商品の説明DisplayLink社製 DL-3100 USB 3.0 グラフィックス・チップセットを採用しています。このPlugable UGA-3000 アダプタを使えば、どの Windows システムにも複数の外部モニタを接続できます（モニタにつき1つのアダプタが必要です）。br複数モニタを 使えば、アプリケーションごとに違う画面で作業することができます。brbr2015 年 3 月販売のモデルより、USB ケーブルが接続されたデザインになっています。brbrb パフォーマンス /bbrbr USBグラフィック・アダプタは、2D/3D画像情報をコンピュータのCPU/GPUによって生成し、圧縮して、変更されたピクセルをUSB経由で転送します。br快適な動作のためには、Dual Core 2Ghz かそれ以上の性能のCPUを推奨いたします。また、USBグラフィックス・アダプタ経由のモニタでは、3Dゲームや動画の再生等ではなく、通常のウェブ表示やアプリケーションで使うことをお勧めしています。brbrb互換性/bbrbrWindows 10、8.1、8、7 の 64 と 32 ビット用のドライバはWindows Updateやダウンロードにより入手できます。brWindows 10 でご使用の場合は DisplayLink 7.9 以上のドライバを使用する必要があります。最新の 7.9 M3 への更新を推奨いたします。br最大解像度が2048x1152 または1920x1200（アナログVGAの場合は最大1920 x1080）までのモニタや、PC接続可能のテレビに接続できます。brbrIntel / AMD / nVidia 社製の、Windows Aeroが使用可能なGPUおよびドライバが必要です。通常の USB グラフィックスの使用には最低Dual Core 2 Ghz 以上の CPU および 2GB RAM をお勧めします。
主な仕様 マルチモニター - このアダプターを使用すると、最高解像度が1920x1200@60Hz までの外部モニタを、Windows システムに接続することができます。モニタ 1 台につき、1 つのアダプタが必要です。DisplayLink チップ DL-3100 を採用し、システムの USB ポート経由で DVI モニタを接続できます。同梱のアダプターを使用して、VGA または HDMI モニターにも接続可能です。（ビデオケーブルは別途必要です。）br生産性の向上 - スクトップ領域を広げて生産性を上げましょう。同時に複数モニタ上に情報を表示することで、より効率的な作業が可能です。br軽量、ポータブル - USB バスパワーで駆動するため電源は必要ありません。トランプカードのようなサイズで持ち運ぶのにも便利です。br互換性 - Windows 11、10、8.x のシステムでで使用することができます。このアダプタを接続する前に、Windows Update を最新状態へ更新することをお勧めします。macOS、Linux、ChromeOS では使用できません。DVI から DisplayPort へ変換をすることはできません。Windows PC の USB-C ポートに接続する場合は、USB ハブまたは A-C ポート変換アダプタ（別売り）を使用してください。br2 年の品質保証 - Plugable は 2 年間の品質保証およびメールでのサポートをご提供いたします。この製品について何かご質問がございましたら、Plugable Japan サポートまでお気軽にご連絡ください。
