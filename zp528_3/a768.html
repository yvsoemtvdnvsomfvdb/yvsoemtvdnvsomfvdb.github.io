国内正規品 Club 3D USB Gen1 Type-C ドッキングステーション 6-in-1 ハブ HDMI 8K30Hz 4K120Hz / 2x USB-A / RJ45 ギガビットイーサネット / USB-C 5Gbps / USB-C PD3.0 100W (CSV-1596)

Club 3D CSV-1596 6-in-1ハブは、Type-Cポートが1ポートのホスト（PC/ラップトップ/モバイルフォンなど）のポートを拡張するための理想的なソリューションを提供します。
USB Type-AおよびType-Cデータポートにより、4台のUSBデバイスを同時に接続することができます。

入力
USB Gen1 Type C オス(ホスト接続)

出力
1x HDMIメス（モニタ接続用）
1x USB タイプAメス（データ）
1x USB タイプAメス（充電/データ）
1x USB タイプCメス（データのみ）
1x USB タイプCメス（PD充電のみ）
1x RJ45ジャック（イーサネット接続用）

長さは約30cm、重さは89gです。

HDMI端子のビデオ出力は、最大7680*4320@30HZ、4196(3840)x2160@120Hz をサポートします。
USB Type C端子は、最大5Gbpsのデータ通信をサポートします。
PD専用のUSB Type C端子は100Wを給電します。ハブ本体で15Wを使用するたホストには最大85Wが給電されます。

全てのOSに対応しています。

ROHS（特定有害物質の使用制限）、FCCとCEマークのEMI要求（人体や他の機器への電磁妨害）に適合しています。

「国内正規保証」。保証期間は購入から２年となります。

モニターの発色具合により実際のものと色が異なる場合がございます。
